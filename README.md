- 👤 用户登录验证系统
- 📊 根据角色展示不同的工作数据
- 🎉 随机祝福语展示
- 📱 响应式设计，支持手机和电脑访问
- ✨ 丰富的动画效果和emoji装饰

## 快速开始

### 方法一：直接打开（电脑端）
直接用浏览器打开 `index.html` 文件即可。

### 方法二：启动服务器（支持手机访问）

1. **安装Python依赖**
   ```bash
   pip install qrcode[pil]
   ```

2. **启动服务器**
   ```bash
   python3 server.py
   ```

3. **访问页面**
   - 电脑端：打开 http://localhost:8000
   - 手机端：扫描终端显示的二维码，或访问显示的IP地址

## 自定义配置

编辑 `bill-config.json` 文件可以修改：
- 用户账号和密码
- 工作数据
- 字段标签和描述
- 祝福语内容

## 海豚姿势和装饰说明

- **登录页面**: 🐬 轻柔漂浮动画
- **工作数据页面**: 根据数据类型显示不同姿势和装饰
  - `working-dolphin`: 🐬⚡ 工作状态（闪电装饰）
  - `coding-dolphin`: 🐬💻 编程状态（电脑装饰）
  - `thinking-dolphin`: 🐬💭 思考状态（思考泡泡装饰）
  - `testing-dolphin`: 🐬🔍 测试状态（放大镜装饰）
- **祝福页面**: 🐬🎉 庆祝跳跃动画（彩带装饰）
